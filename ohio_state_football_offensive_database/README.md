# Ohio State Football Offensive Database

**Independent Study Project** | Spring 2026 | [Back to Portfolio](../)

This project builds a structured database for Ohio State football offensive data. The goal is to organize offensive information in a way that supports analysis, filtering, and future modeling work.

## Key Features
- Stores offensive football data in a clean and usable format.
- Supports player evaluation and offensive trend analysis.
- Creates a foundation for future football analytics projects.
